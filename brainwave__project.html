Brainwave Machine Learning

Project Overview

Current text-based authentication systems cannot withstand the situation where the user is forced to release the password under hostile circumstances. An alternative is Coercion Resistant authentication systems which generate passwords that cannot be explained to others, will not be authenticated under pressure, and are not susceptible to brute-force attacks.

The PolySec Lab is working on creating an unbreakable password based on brain signals. If the password is set using brainwaves, at what point will your brainwaves be viable to open a system when you are being held against your will?

Current text-based authentication systems are vulnerable when users are forced to reveal their passwords under duress. A better alternative is Coercion Resistant Authentication, which generates passwords that cannot be explained to others, withstand pressure, and resist brute-force attacks. This research explores a specific implementation using an individual's subconscious physiological and neurological responses to music as an authenticating factor.

During training, the user's baseline physiological responses to personally chosen, stimulating music are recorded multiple times. For authentication, the same music is played, and the user's responses are compared to the stored data. Authentication validity is determined by the similarity of the new and training responses, demonstrating that certain music can create reliable Coercion Resistant Authentication.

Electroencephalography (EEG) data, which describe neuron activity in the brain and are acquired via skull electrodes, are unique to individuals and suitable for authentication. The advent of consumer-grade EEG devices has made EEG research more accessible for use in smartphones, but current software lacks seamless data collection and analysis for Android.

EEG Workbench (EEGW) addresses this gap, providing developers with a tool to facilitate the development of EEG-based recognition systems for computational neuroscience and brain-computer interface projects.
